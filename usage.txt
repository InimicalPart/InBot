You need node version 16 or above and npm version 6.14.13

To change node version, install nvm: https://github.com/coreybutler/nvm-windows/releases/
then when done installing, open cmd and type:
- nvm install 16.6.2
- nvm use 16.6.2

To change npm version, open cmd and type:
- npm install -g npm@6.14.13
